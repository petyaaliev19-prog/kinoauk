# Киноаук

Мини-приложение для выбора одного фильма колесом.

## Как запустить

Лучше запустить локально:

```powershell
node server.js
```

Потом открыть `http://127.0.0.1:5173`.

Через `index.html` тоже можно, но быстрый импорт с Кинопоиска надежнее работает через локальный адрес.

## Самый простой импорт “Хочу посмотреть” с Кинопоиска

Самый удобный вариант - кнопка `Обновить с Кинопоиска` в самом Киноауке.

1. Запусти `start-kinoauk.cmd` или ярлык `Киноаук Максима и Оли` на рабочем столе.
2. Откроется `http://127.0.0.1:5173`.
3. Во вкладке `Импорт` нажми `Обновить с Кинопоиска`.

Терминал с сервером должен оставаться открытым, пока пользуешься Киноауком.

Чтобы остановить сервер, нажми кнопку `⏻` в правом верхнем углу сайта, закрой терминал или запусти `stop-kinoauk.cmd`.

Ссылки на списки уже зашиты в проект:

`https://www.kinopoisk.ru/user/14758743/movies/planned-to-watch/`

`https://www.kinopoisk.ru/user/33826991/movies/planned-to-watch/`

Импорт проходит страницы списков автоматически и собирает все найденные фильмы и сериалы.

## Запасной импорт через буфер

1. Открой на Кинопоиске свой список `Хочу посмотреть`.
2. Пролистай страницу вниз, чтобы подгрузились нужные фильмы.
3. Нажми `Ctrl+A`, потом `Ctrl+C`.
4. Вернись в Киноаук, открой вкладку `Импорт`.
5. Нажми на большую область импорта и сделай `Ctrl+V`.

Киноаук сам вытащит из буфера ссылки на фильмы и добавит их в колесо.

## Импорт через закладку

1. В Киноауке открой вкладку `Импорт`.
2. Перетащи кнопку `Импорт в Киноаук` на панель закладок.
3. Открой на Кинопоиске свой список `Хочу посмотреть`.
4. Пролистай страницу вниз, чтобы подгрузились нужные фильмы.
5. Нажми закладку `Импорт в Киноаук`.

Киноаук откроется обратно и сам добавит фильмы в колесо.

Если Киноаук открыт как файл, закладка скопирует обычный список в буфер. Тогда вернись в Киноаук и нажми `Из буфера`.

Еще можно вставить обычный список названий или ссылок во вкладке `Импорт`.
